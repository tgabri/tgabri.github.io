Final Project

The Food Miles mobile app, five Northcoders including myself worked on in our final 2 weeks, user must take a photo of a product's label, the app can recognise the origin of the product and show the user how far food has travelled before it reaches the consumer. RESTful API built with Node.js, Express.js, MongoDB and front end built with React Native. Testing carried out using Mocha, Chai and Supertest.
